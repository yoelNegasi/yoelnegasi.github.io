Random Number

The Ranmdom Number Game is a dice game where you can try to guess a random number from 1 to 6.The one who randomly chooses number 6 adds a point to his score.This project was created by HTML CSS and Javascript. This game can be played with two people, if a player get's the number six player gets another chance until he no longer gets number six.Its really fun to play with friends.
